    Scientific Calculator

    This is a browser based calculator built using HTML, CSS, and JavaScript. It allows users to perform basic and scientific calculations with responsive buttons and real time results. The layout is styled with a soft pink theme, rounded buttons and dark mode toggle. Users can input functions like sin, cos or ^ and manually add brackets to complete expressions. All calculations happen in the browser, making it a fully frontend project. It includes a back button to review previous inputs and smooth animation effects improve the users interaction.

    To-Do List

    The To-Do List is a task management tool created using HTML, CSS and JavaScript. Users can add, edit or delete tasks easily with clickable icons. When a task is marked as done, it's struck through and a progress bar updates to show completion percentage. The design uses soft pink and white colors with a clean layout. Time and date are displayed for each task. All data is stored in memory. This To-Do List is a frontend project that is perfect for organizing daily routines efficiently since it improves productivity by keeping everything clear.
